## 第27课：堆结构常见题型

**题目描述：**

给定 k 个已排序的链表，将它们合并成一个排序后的链表。

**输入描述：**

- 第一行一个整数 k，表示有 k 个链表。
- 接下来 k 行，每行包含若干个空格隔开的整数，表示每个链表的元素。

**输出描述：**

- 输出一个链表，包含所有输入链表中的元素，并且按升序排列。

**示例：**

输入：

```
2
1 2 3
4 5 6 7
```

输出：

```
1 2 3 4 5 6 7
```

```c++
/**
 * struct ListNode {
 *	int val;
 *	struct ListNode *next;
 *	ListNode(int x) : val(x), next(nullptr) {}
 * };
 */
class Solution {
public:
    // 合并k个有序链表的函数
    ListNode* mergeKLists(vector<ListNode*>& arr) {
        // 小根堆，用于存储链表节点
        priority_queue<ListNode*, vector<ListNode*>, function<bool(ListNode*, ListNode*)>> heap(
        [](ListNode* a, ListNode* b) { return a->val > b->val; });

        // 将所有非空链表的头节点加入小根堆
        for (ListNode* head : arr) {
            if (head != nullptr) {
                heap.push(head);
            }
        }

        // 如果堆为空，返回空指针（没有链表需要合并）
        if (heap.empty()) {
            return nullptr;
        }

        // 弹出堆顶的最小节点作为合并链表的头节点
        ListNode* head = heap.top();
        heap.pop();
        ListNode* pre = head;

        // 如果弹出的节点有下一个节点，将其加入堆中
        if (pre->next != nullptr) {
            heap.push(pre->next);
        }

        // 处理堆中剩余的节点
        while (!heap.empty()) {
            ListNode* cur = heap.top();
            heap.pop();
            pre->next = cur;  // 将当前最小节点连接到合并链表中
            pre = cur;        // 更新 "pre" 指针为当前节点

            // 如果当前节点有下一个节点，将其加入堆中
            if (cur->next != nullptr) {
                heap.push(cur->next);
            }
        }

        return head;  // 返回合并后链表的头节点
    }
};
```
